## 🚀 部署和使用

### 本地运行
```bash
# 安装依赖
pip install -r requirements.txt

# 启动系统
python start.py

# 访问地址
http://localhost:5000
```

### GitHub部署
1. 使用 `upload_to_github.bat` 脚本
2. 或按照 `GITHUB_SETUP.md` 手动操作
3. 推荐仓库名: `fruit-vegetable-ai-service`
