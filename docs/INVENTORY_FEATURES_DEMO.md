## 🎯 核心功能演示

### 产品入库流程演示
```
步骤1: 访问产品入库页面
→ 点击侧边栏"📦 产品入库"

步骤2: 填写产品信息
→ 产品名称: "红富士苹果"
→ 产品分类: "水果"
→ 价格: "15.8元/斤"
→ 单位: "斤"

步骤3: 实时条形码生成
→ 系统自动生成条形码: 880000614337
→ 显示条形码图片预览
→ 产品ID: P614337

步骤4: 完善其他信息
→ 存储区域: "A区 - 水果区"
→ 初始库存: 100
→ 最小库存警告: 10

步骤5: 保存产品
→ 点击"保存产品"按钮
→ 显示成功提示
→ 条形码图片保存到static/barcodes/
```

### 库存盘点流程演示
```
步骤1: 创建盘点任务
→ 点击"+ 创建新盘点"
→ 输入备注: "月度盘点"
→ 生成任务ID: COUNT_20250610_143022_abc123

步骤2: 添加盘点产品
→ 方式1: 输入条形码 "880000614337"
→ 方式2: 搜索产品名称 "苹果"
→ 系统自动添加到盘点列表

步骤3: 录入实际数量
→ 账面数量: 100 (系统自动获取)
→ 实际数量: 95 (手动输入)
→ 差异: -5 (自动计算，红色显示)

步骤4: 完成盘点
→ 确认所有产品已录入
→ 查看汇总: 总项目3个，有差异3个
→ 点击"完成盘点"
→ 生成盘点报告
```

### 数据对比分析演示
```
步骤1: 选择分析类型
→ 选择"周对比分析"
→ 点击"生成周对比"
→ 系统自动选择最近两次盘点

步骤2: 查看分析结果
→ 统计汇总: 总产品25个，变化8个，异常3个
→ 变化明细: 显示每个产品的具体变化
→ 异常检测: 标记超出阈值的变化项目

步骤3: 生成报告
→ 点击"📄 下载分析报告"
→ 生成Markdown格式报告
→ 包含详细分析和管理建议

步骤4: 导出数据
→ 点击"📊 导出Excel"
→ 生成CSV格式数据文件
→ 用于进一步分析处理
```

## 📊 数据流演示

### 产品数据流
```
产品录入 → 条形码生成 → 数据验证 → 保存到inventory.json
    ↓
库存调整 → 历史记录 → 操作日志 → 低库存检查
    ↓
盘点任务 → 差异计算 → 盘点报告 → 库存更新
    ↓
对比分析 → 趋势分析 → 异常检测 → 管理建议
```

### API调用流程
```
前端表单提交
    ↓
JavaScript验证
    ↓
AJAX请求到Flask API
    ↓
后端业务逻辑处理
    ↓
数据存储到JSON文件
    ↓
返回JSON响应
    ↓
前端更新界面显示
```
